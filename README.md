# Best Nonogram
This C# program help you do the puzzles that will earns you the most XP on nonogram katana. It will filter out puzzles that you made in the last 31 days and sort them for XP. Here is a short usage guide

- Copy paste the name of the puzzle you wish to solve into the game
- Solve the nonogram
- Enter the name of the puzzle you just did in the terminal
- Press enter

If you wish to update the list of puzzles, you can use https://addons.mozilla.org/en-US/firefox/addon/tbl2csv/ and do those steps : Right click -> "Table to CSV" -> "HTML content" for those two tables :
- https://nonograms-katana.fandom.com/wiki/High_XP_puzzle_collection#Color_nonograms
- https://nonograms-katana.fandom.com/wiki/High_XP_puzzle_collection#Black-and-white_nonograms
