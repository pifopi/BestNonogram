# Best Nonogram
This C# program help you do the puzzles that will earns you the most XP on nonogram katana. It will filter out puzzles that you made in the last 31 days and sort them for XP.

If you wish to update the list of puzzles, you can use your browser's html inspector on those pages to copy paste it from those pages (in lua format)
